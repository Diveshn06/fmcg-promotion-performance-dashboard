# FMCG Promotion Performance Dashboard

## Project Overview
This project analyzes promotional campaigns conducted by AtliQ Mart across multiple stores during festive seasons.

## Tools Used
Power BI
SQL
DAX
Data Modeling

## Key Insights
• Cashback promotions generated highest incremental revenue  
• BOGOF drove maximum unit sales  
• 25% discount campaigns negatively impacted revenue  
• Diwali campaigns outperformed Sankranti

## Skills Demonstrated
Star Schema Modeling  
Advanced DAX  
Business KPI Tracking  
Interactive Dashboard Design
